WELCOME TO STRESSBRIDGE

Here are a few instructions on how to run the game:
1) Download the game zip provided
2) Extract the zip into a folder anywhere on your desktop
3) Run server.py on the server machine	
4) Run client.py on the client machines

---For First Time Login---
5) Enter username of your choosing
6) Set password of your liking and the game will start

---For Consecutive Login---
7) Enter your previous username and key in your password
